# Test Files

This directory contains various test files and utilities for the Pocket Prompt UI application.

## Deep Link Testing

Located in `deep-links/` subdirectory:

- `debug-deep-link.sh` - Shell script for debugging deep link functionality
- `test-deep-link-application.js` - Node.js script to test deep link handling
- `test-deep-link.html` - HTML page for testing web deep links
- `test-deep-link.js` - JavaScript utilities for deep link testing
